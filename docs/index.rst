Welcome to log4data's documentation!
====================================

log4data is a Python library designed to simplify the logging setup process for new data pipelines. It provides a straightforward way to configure loggers, eliminating the repetitive task of setting up logging configurations. With log4data, you can focus on building your data pipelines without worrying about the intricacies of logging.

Whether you're looking for a quick setup with default configurations or need more advanced features like parameterized file names and logging levels, log4data offers a flexible and user-friendly approach to meet your logging needs.
